# Project Delivery App

# Contexto
Este projeto trata-se de uma uma plataforma de delivery de cerveja.
Nessa aplicação, o grupo foi responsável por criar e integrar tanto o back-end quanto o front-end.

## Técnologias usadas

Front-end:
> Desenvolvido usando: React, CSS3, HTML5, ES6

Back-end:
> Desenvolvido usando: NodeJS, ExpressJS, MYSQL, ES6


## Instalando Dependências

> Backend
```bash
cd api/ 
npm install
``` 
> Frontend
```bash
cd src/
npm install
``` 
## Executando aplicação

* Para rodar o back-end:

  ```
  cd api/ && npm start
  ```
* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```

## Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
